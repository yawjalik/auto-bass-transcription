# Client (Frontend)

Built with Next.js 14

## How to run

1. Install dependencies

    ```bash
    npm install
    ```

    or

    ```bash
    yarn
    ```

2. Run the development server

    ```bash
    npm run dev
    ```

    or

    ```bash
    yarn dev
    ```

3. Build and run

    ```bash
    npm run build
    npm run start
    ```

    or

    ```bash
    yarn build
    yarn start
    ```
